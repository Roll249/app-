# Changelog

## [1.0.0] - 2026-04-11

### Added
- Tạo mới project personal-finance-app
- Backend Kotlin/Ktor với đầy đủ API endpoints
- 10 ngân hàng ảo Việt Nam
- Hệ thống QR VietQR
- Android Client với Clean Architecture + MVI
- Room Database cho offline-first
- Retrofit cho network layer
- Material3 UI với Jetpack Compose

### Backend Features
- Authentication (JWT + BCrypt)
- Account management
- Transaction management
- Fund (Savings goals)
- Budget tracking
- Financial reports
- Bank transfer simulation
- QR code generation/validation

### Android Features
- Login/Register screens
- Home dashboard
- Account list/detail
- Transaction list/add
- Fund management
- Bank list (10 banks)
- Budget tracking
- Report dashboard
- QR generate/scan
- Profile & Settings

### Technical
- Kotlin 2.0
- Ktor 2.3.9
- Room 2.6.1
- Hilt 2.50
- Retrofit 2.9.0
- Navigation Compose 2.7.7
